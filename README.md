# KaypiAdmin

Crear la imagen:
docker-compose build

Iniciar contenedor:
docker-compose up

Crear .tar:
docker build -t  kaypidocker .
docker save -o docker_admin.tar kaypidocker
Siendo kaypidocker el nombre de la imagen
